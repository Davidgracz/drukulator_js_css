1. Utwórz repozytorium na GitHubie.
2. Wgraj wszystkie pliki z tego folderu do głównego katalogu repozytorium.
3. Wejdź w Settings -> Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: / (root).
6. Kliknij Save.
7. Otwórz adres podany przez GitHub Pages.

Ceny edytuje się w pliku data/prices.json.
Projekt nie wymaga Pythona ani Streamlit.
